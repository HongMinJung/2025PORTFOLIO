# 풀스택 개발자 포트폴리오 사이트

Next.js 15, React 19, TypeScript, TailwindCSS 4, Three.js를 활용한 개인 포트폴리오 사이트입니다.

## 주요 기능

- **Three.js 기반 인터랙티브 히어로 섹션** - 3D 그래픽과 애니메이션
- **반응형 디자인** - 모든 디바이스에서 최적화된 경험
- **다크/라이트 모드** - 사용자 환경 설정 지원
- **프로젝트 갤러리** - 필터링 및 검색 기능
- **기술 스택 시각화** - 개발자 스킬 프로필 제공
- **연락 폼** - 이메일 메시지 전송 기능

## 시작하기

1. 저장소 클론:

```bash
git clone https://github.com/HongMinJung/2025PORTFOLIO
cd 2025PORTFOLIO
```

2. 의존성 설치:

```bash
npm install
```

3. 개발 서버 실행:

```bash
npm run dev
```

4. 브라우저에서 [http://localhost:3000] 접속

## 배포

Vercel을 통해 자동으로 배포합니다.

```bash
npm run build
```

생성된 빌드를 확인하려면:

```bash
npm run start
```
